What is Kafka?

Apache Kafka is a distributed event streaming platform. Originally built at LinkedIn and open-sourced under the Apache Software Foundation, it's designed to handle high-throughput, fault-tolerant, real-time data feeds.

Core Concepts

Event

An event records the fact that "something happened" in your system. Every event has a key, a value, a timestamp, and optional metadata (headers). In Kafka, an event is also called a message or record. 

Event {
  Key:       "user-42"
  Value:     "{\"action\":\"page_view\",\"page\":\"/pricing\"}"
  Timestamp: 2026-04-28T20:00:00Z
  Headers:   { "source": "web-app" }
}

Topic

Events are organized into topics. A topic is a named category or feed — think of it like a table in a database or a folder in a filesystem. Producers write events to topics; consumers read events from topics.

Partition

Topics are split into partitions. Each partition is an ordered, immutable sequence of events. Partitions are what give Kafka its scalability — different partitions can live on different brokers and be consumed in parallel.

Broker

A Kafka broker is a server that stores data and serves clients. A cluster is a group of brokers working together. A common production setup has 3+ brokers for fault tolerance.

Why Use Kafka?

  • High throughput — millions of events per second on modest hardware
  • Durability — data is persisted to disk and replicated across brokers
  • Scalability — add brokers to handle more data without downtime
  • Decoupling — producers and consumers don't need to know about each other
  • Stream processing — process events in real-time with Kafka Streams or ksqlDB
Tip Kafka is not a traditional message queue. Unlike RabbitMQ or ActiveMQ, Kafka keeps events around even after they're consumed (configurable retention). This makes it great for replaying data and building event-driven architectures.

Kafka Ecosystem

Component What it does
Kafka Broker Stores and serves events
Kafka Connect Stream data between Kafka and external systems (DBs, S3, etc.)
Kafka Streams Lightweight stream processing library for Java/Scala apps
ksqlDB Stream processing with SQL syntax
Schema Registry Central repository for Avro/Protobuf/JSON schemas
